# ona_announcement_banner (Resource)

Singleton Ona announcement banner shown on the organization home page. Organization scope is resolved from the configured provider token. Destroying this resource disables and clears the remote banner because the Ona API has no delete operation.

For product context, see [Home notice](https://ona.com/docs/ona/organizations/announcement-banner).

## Example Usage

```terraform
resource "ona_announcement_banner" "example" {
  enabled = true
  message = "Scheduled maintenance on Saturday 22:00-02:00 UTC."
}
```

## Schema

### Required

- `enabled` (Boolean) Whether the announcement banner is displayed to organization members.

### Optional

- `message` (String) Announcement banner body. Supports basic Markdown and must be at most 1000 characters. Required when `enabled` is true.

### Read-Only

- `id` (String) Authenticated organization ID for the managed announcement banner singleton.
