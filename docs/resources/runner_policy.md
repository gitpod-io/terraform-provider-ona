# ona_runner_policy (Resource)

Ona runner policy granting a group access to a runner.

This provider page is the current product reference for runner policies. Review the generated schema and [Organization roles](https://ona.com/docs/ona/organizations/organization-roles) before granting automation credentials permission to manage runner access.

## Example Usage

```terraform
resource "ona_runner_policy" "org_members" {
  runner_id = ona_runner.example.runner_id
  group_id  = ona_group.example.id
  role      = "user"
}
```

## Schema

### Required

- `group_id` (String) Group ID receiving access to the runner.
- `runner_id` (String) Runner ID this policy grants access to.

### Optional

- `role` (String) Runner role granted to the group. The only supported value is `user`.

- `id` (String) Terraform resource ID in `runner_id/group_id` format.
