# ona_integration_definitions (Data Source)

Fetches all integration definitions visible to the configured provider token. Definitions are global templates managed by Ona; Terraform can discover but not mutate them. Experimental definitions are filtered by backend visibility rules.

For product context, see [Integrations](https://ona.com/docs/ona/integrations/overview) and [MCP servers](https://ona.com/docs/ona/mcp).

## Example Usage

```terraform
data "ona_integration_definitions" "available" {}

locals {
  integration_definitions_by_id = {
    for definition in data.ona_integration_definitions.available.definitions :
    definition.id => definition
  }
}
```
